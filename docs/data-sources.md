# 数据来源文档

本文档描述项目使用的数据接口、CSV存储以及akshare/efinance API调用详情。

## 数据来源概览

| 数据类型 | 数据源 | CSV存储文件 | 主要用途 |
|---------|--------|-----------|---------|
| 红利指数持仓 | akshare - 中证指数 | `红利指数持仓汇总.csv` | 获取红利类指数成分股 |
| 股票分红次数 | akshare - 历史分红 | `股票分红次数汇总.csv` | 筛选历史分红次数多的股票 |
| 个股板块映射 | efinance | `个股板块映射.csv` | 获取股票概念板块和行业板块 |
| M120均线数据 | akshare - 历史K线 | `M120数据.csv` | 计算120日均线和偏离度 |
| PE/PB数据 | akshare - 实时行情 | `PE数据.csv` | 获取估值指标和市值 |
| 实时股价 | akshare - 分时数据 | 无（实时查询） | 获取最新收盘价 |

## 1. 红利指数持仓数据

### 1.1 数据接口

**数据源**: akshare

**调用位置**: `src/data/fetcher.py` - `IndexHoldingsFetcher.fetch_index_holdings()`

**API调用**:
```python
ak.index_stock_cons_weight_csindex(symbol=index_code)
```

**参数说明**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| symbol | str | 是 | 指数代码，如 "000922" |

**支持的指数**:
| 指数代码 | 指数名称 |
|---------|---------|
| 000922 | 中证红利 |
| 932315 | 中证红利质量 |
| 932309 | 红利增长 |
| 931468 | 红利质量 |

**返回字段** (原始返回):
| 字段 | 说明 |
|------|------|
| 成分券代码 | 股票代码 |
| 成分券名称 | 股票名称 |
| 权重 | 在指数中的权重 |

### 1.2 CSV存储

**文件**: `data/红利指数持仓汇总.csv`

**字段说明**:
| 字段 | 类型 | 说明 |
|------|------|------|
| 交易所 | str | SH（沪市）/SZ（深市） |
| 股票代码 | str | 6位股票代码 |
| 股票名称 | str | 股票简称 |
| 来源指数 | str | 所属的红利指数名称 |
| 来源指数代码 | str | 所属的红利指数代码 |
| 纳入指数数量 | int | 该股票在几个红利指数中 |

## 4. M120均线数据

### 4.1 数据接口

**数据源**: akshare

**调用位置**: `src/services/m120_service.py` - `M120Service.calculate_m120()`

**API调用**:
```python
ak.stock_zh_a_hist(symbol=code, period="daily", adjust="qfq")
```

**参数说明**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| symbol | str | 是 | 股票代码，6位数字 |
| period | str | 否 | 周期，默认 "daily"（日线） |
| adjust | str | 否 | 复权方式，"qfq"（前复权） |

**返回字段** (原始返回):
| 字段 | 说明 |
|------|------|
| 日期 | 交易日期 |
| 开盘 | 开盘价 |
| 收盘 | 收盘价 |
| 最高 | 最高价 |
| 最低 | 最低价 |
| 成交量 | 成交量 |
| 成交额 | 成交额 |
| 振幅 | 振幅 |
| 涨跌幅 | 涨跌幅 |
| 涨跌额 | 涨跌额 |
| 换手率 | 换手率 |

### 4.2 计算逻辑

1. 获取最近120天的收盘价数据
2. 计算120日均线: `MA120 = 最近120天收盘价平均值`
3. 获取最新收盘价
4. 计算偏离度: `偏离度(%) = (最新收盘价 - MA120) / MA120 * 100`

### 4.3 CSV存储

**文件**: `data/M120数据.csv`

**字段说明**:
| 字段 | 类型 | 说明 |
|------|------|------|
| 股票代码 | str | 6位股票代码 |
| M120 | float | 120日均线值 |
| 收盘价 | float | 最新收盘价 |
| 偏离度 | float | 收盘价相对M120的偏离度(%) |

## API接口与数据映射

### API路由

| 路由 | 方法 | 数据源 | CSV读取/写入 |
|------|------|--------|-------------|
| `/m120` | GET | 读取 `M120数据.csv` | 读取 |
| `/m120/refresh` | POST | akshare历史K线 | 写入 `M120数据.csv` |
| `/pe` | GET | 读取 `PE数据.csv` | 读取 |
| `/pe/update` | POST | akshare实时行情 | 写入 `PE数据.csv` |
| `/realtime-price` | POST | akshare分时数据 | 无 |
| `/stocks/info` | POST | 读取 `个股板块映射.csv` | 读取 |
| `/stocks` | GET | 读取多个CSV | 读取 |
| `/stocks/{code}` | GET | 读取多个CSV | 读取 |

## 数据更新频率建议

| 数据类型 | 建议更新频率 | 说明 |
|---------|-------------|------|
| 红利指数持仓 | 每月/季度 | 指数成分股变动较少 |
| 股票分红次数 | 每半年/年度 | 历史数据变化较慢 |
| 个股板块映射 | 每季度 | 板块归属变化较慢 |
| M120均线数据 | 每日 | 交易收盘后更新 |
| PE/PB数据 | 每日 | 交易收盘后更新 |
| 实时股价 | 实时 | 用户查询时获取 |
