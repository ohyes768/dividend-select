## 一、项目概览

**项目名称**：A股高股息率TOP50查询工具

**核心功能**:筛选沪深主板市场的高股息率股票，展示分红历史、股价趋势、行业分类等信息。

**技术栈**:
- 语言: Python 3.11+
- 依赖管理: uv + pyproject.toml
- 数据源: akshare（财经数据）、efinance（板块信息）
- 数据处理: pandas

## 五、统一入口 main.py

**命令行参数**:
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--use-local` | 使用本地已有数据，跳过API获取 | False |
| `--limit N` | 限制处理的股票数量（测试用） | 0（不限制） |
| `--min-dividend N` | 最小分红次数阈值 | 5 |

**运行流程**:
```
main()
├── parse_args()                    # 解析命令行参数
├── Step 1: IndexHoldingsFetcher.get_stock_list()
├── Step 2: BoardMappingFetcher.update()  # 仅默认模式，更新板块映射
├── Step 3: load_existing_codes()   # 检查已处理的股票，实现断点续传
├── Step 4: BoardInfoLoader.get_all_board_info()  # 预加载板块信息
├── Step 5: DividendCalculator.calculate_all()  # 增量写入CSV
└── Step 6: display_summary()       # 显示汇总信息
```

**断点续传功能**:
- 程序启动时会检查 `近3年股息率汇总.csv` 中已存在的股票代码
- 自动跳过已处理的股票，只处理剩余股票
- 每计算完一只股票，立即追加写入CSV（增量写入）
- 即使程序中断，已处理的数据不会丢失
