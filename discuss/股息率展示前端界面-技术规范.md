# 股息率展示前端界面 - 技术规范

## 1. 概述

### 1.1 背景

现有 A 股高股息率查询工具是一个纯 Python 后端项目，能够生成 `近3年股息率汇总.csv` 数据文件。用户需要通过 Web 界面可视化展示这些数据，方便筛选和查看股票的股息率信息。

### 1.2 目标

**核心目标：**
- 实现 Web 前端界面，展示近3年股息率数据
- 支持按股息率阈值筛选
- 支持按股息率排序
- 支持行展开查看季度股息率详情

**附加目标：**
- 使用虚拟滚动优化性能
- 使用 Docker Compose 统一部署

**非目标：**
- 不支持用户注册/登录
- 不支持数据修改
- 不支持实时数据推送（CSV更新后手动刷新）

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 股息率阈值筛选 | 用户可自定义阈值，默认 3% | P0 |
| 数据表格展示 | 虚拟滚动，不分页 | P0 |
| 按股息率排序 | 支持升序/降序切换 | P0 |
| 行展开功能 | 展开后显示季度股息率详情 | P0 |

### 2.2 用户故事

**场景1：筛选高股息率股票**
用户想要找到股息率高于 4% 的股票，输入阈值 4%，点击筛选，表格自动更新显示符合条件的股票。

**场景2：查看季度详情**
用户点击某行股票的展开按钮，查看该股票 2025 年 Q1-Q4 的季度股息率详情，了解分红的时间分布。

## 3. 技术决策

### 3.1 技术栈

| 类别 | 技术选型 | 理由 |
|------|---------|------|
| 前端框架 | Next.js v15.4 | 现代全栈框架，内置 SSR 和路由 |
| UI 框架 | React v19 | 最新版本，性能优化 |
| 样式方案 | Tailwind CSS v4 | 原子化 CSS，快速开发 |
| 类型系统 | TypeScript | 强类型，避免运行时错误 |
| 虚拟滚动 | @tanstack/react-virtual | 高效的虚拟滚动实现 |
| 后端框架 | FastAPI | Python 原生支持，类型安全 |
| 部署方案 | Docker Compose | 统一编排，简化部署 |

**系统架构：**

**模块划分：**

- `api/` - API 路由和模型定义
- `services/` - 业务逻辑服务
- `config/` - 配置管理

**前端 (frontend/)**
- `app/` - Next.js 页面
- `components/` - React 组件
- `hooks/` - 自定义 Hook
- `lib/` - 工具函数

### 3.3 方案权衡记录

**虚拟滚动 vs 分页**
- 考虑：数据量约 188 条
- 选择：虚拟滚动
- 理由：用户可以快速浏览全部数据，无需翻页，体验更流畅

**后端排序 vs 前端排序**
- 考虑：数据量不大
- 选择：后端排序
- 理由：数据源统一，未来数据量增长时性能更好

**手动刷新 vs 自动更新**
- 考虑：CSV 文件更新频率不高
- 选择：手动刷新
- 理由：实现简单，避免轮询带来的服务器压力

## 4. 数据设计

### 4.1 数据结构

#### 股票数据模型

```typescript
interface DividendStock {
  // 基础信息
  code: string;
  name: string;
  exchange: string;

  // 股息率数据
  avgYield3y: number | null;
  yield2025: number | null;
  yield2024: number | null;
  yield2023: number | null;

  // 价格与分红
  avgPrice3y: number | null;
  dividend2025: number | null;
  dividendCount2025: number | null;

  // 价格波动
  highPrice2025: number | null;
  lowPrice2025: number | null;
  highChangePct2025: number | null;
  lowChangePct2025: number | null;

  // 季度数据
  quarterly: QuarterlyData;
}
```

#### 季度数据模型

```typescript
interface QuarterlyData {
  q1: { avgPrice: number | null; dividend: number | null; yield: number | null; };
  q2: { avgPrice: number | null; dividend: number | null; yield: number | null; };
  q3: { avgPrice: number | null; dividend: number | null; yield: number | null; };
  q4: { avgPrice: number | null; dividend: number | null; yield: number | null; };
}
```

## 7. 边缘情况与错误处理

### 7.1 异常场景

**网络异常：**
- 显示友好的错误提示
- 提供重试按钮

**数据异常：**
- 空值显示为 "-"
- 无效数据过滤

### 7.2 边界条件

**阈值输入：**
- 负数：自动修正为 0
- 超大值：限制最大值

**数据量为 0：**
- 显示暂无数据提示

### 7.3 降级策略

- API 不可用：显示缓存数据或友好提示
- CSV 文件缺失：返回 404 错误
