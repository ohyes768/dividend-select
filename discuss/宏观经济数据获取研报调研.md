## 一、EPFR 资金流向数据获取

### 1.1 直接接入 EPFR 官方

| 项目 | 说明 |
|------|------|
| **数据源** | EPFR Global (eVestment) 是全球领先的资金流向数据提供商 |
| **获取方式** | 官方数据终端订阅服务 |
| **准入门槛** | 需要 **企业级授权**，个人用户较难直接接入 |
| **费用** | 通常费用较高，需要商务谈判 |
| **中国接入方式** | 大多数国内券商（如中信证券）都是通过 EPFR 授权获取数据，再转发给客户 |

### 1.2 通过国内数据服务商

| 服务商 | 状况 |
|--------|------|
| Wind (万得) | 有 EPFR 数据，但需要付费终端 |
| Choice (东方财富) | 可能包含部分资金流向数据 |
| Tushare | 主要是国内 A 股资金流向，无 EPFR |
| 聚宽/米筐等 | 量化平台，可能有相关数据 |

### 1.3 替代数据源

- **北向资金/南向资金**: 通过沪深港通数据获取，多家平台提供免费 API
- **基金净值数据**: 可通过基金公司官网或数据平台获取，间接推断资金流向

## 二、中金全球资金流向周报获取

### 2.1 获取途径

| 途径 | 难度 | 备注 |
|------|------|------|
| 官方研报系统 | 高 | 需要中金客户权限 |
| 券商研报平台 | 中 | 如慧科、万得、东方财富研报库 |
| PDF 自动化解析 | 低 | 技术上可行，但需注意版权 |

### 2.2 技术实现建议

**方案 A - 第三方数据服务 (推荐)**
```
Wind API → 提供资金流向数据
Choice API → 可能有全球资金流向
```

**方案 B - 替代数据源**
```
Tushare Pro → 北向/南向资金
iFinD (同花顺) → 部分资金流向数据
```

**方案 C - 研报数据提取 (需注意合规)**
- 使用 OCR 或 PDF 解析库
- 从公开研报中提取数据
- ⚠️ 需注意版权和合规问题

### 2.3 推荐优先级

1. **Tushare Pro** - 免费/低门槛，有资金流向相关数据
2. **Wind API** - 数据最全，但费用高
3. **北向/南向资金 API** - 免费且数据相对准确
4. **研报数据提取** - 作为补充，需谨慎处理版权

## 三、宏观经济研报推荐

### 3.1 国内券商（第一梯队）

| 券商 | 宏观团队 | 特色 |
|------|---------|------|
| **中金公司** | 彭文生、黄海洲 | 顶尖梯队，深度研究 |
| **中信证券** | 褚建芳、明明 | 数据详实，覆盖全面 |
| **国泰君安** | 花长春 | 产业与宏观结合好 |

| **申万宏源** | 李慧勇 | 指标分析细致 |

### 3.2 其他值得关注

- **招商证券** - 谢亚轩
- **兴业证券** - 王涵
- **广发证券** - 郭磊

### 3.3 国际投行

| 机构 | 价值 |
|------|------|
| **高盛** | 全球视野，对中国影响判断准 |
| **摩根士丹利** | 深度专题多，逻辑清晰 |
| **瑞银 (UBS)** | 欧洲视角，参考价值高 |
| **花旗** | 跨国比较做得好 |

### 3.4 官方权威报告（必看）

#### 央行系列
- **货币政策执行报告**（每季度）- 核心中的核心
- **金融稳定报告**（年度）
- **货币政策委员会例会纪要**

#### 统计局系列
- **季度/年度 GDP 数据**
- **CPI/PPI 月度数据**
- **PMI 制造业/非制造业**

#### 其他官方
- **财政部：财政收支报告**
- **海关总署：进出口数据**
- **外管局：外汇储备数据**

### 3.5 重点关注指标

| 类别 | 核心指标 |
|------|---------|
| 产出 | GDP、工业增加值 |
| 需求 | 社零、固定资产投资、出口 |
| 价格 | CPI、PPI、核心CPI |
| 金融 | 社融、M2、信贷 |
| 就业 | 城镇调查失业率 |
| 领先 | PMI、发电量、运价 |

### 3.6 获取渠道

| 渠道 | 说明 |
|------|------|
| Wind/Choice | 最全但收费 |
| 东方财富/同花顺 | 免费但有延迟 |
| 券商 App | 开户客户可看 |
| 慧科 | 研报聚合平台 |
